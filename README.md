# Petrichor_CN
屋子补丁破解

ペトリコール -Petrichor-  
雨后泥土的气息 -Petrichor-  

# 用法
你有两种可选  
第一种是直接下载 petrichor_cn.exe 放到游戏目录下运行即可  
第二种是把file文件夹里的全部文件拷贝到游戏目录下运行 petrichor_cn.exe  

区别就是  
第一个方法是打包好的单文件  
第二个是没打包的零碎文件  
因为evb打包对这种小文件数量特别多的效率很差  
打包的会卡一点。  

# 注意
这个补丁仅仅面向非大陆地区  
如果你在大陆地区传播该补丁出问题你自己担着。
